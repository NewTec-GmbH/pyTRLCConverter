get_table_options(trlc_package: str, trlc_type: str, trlc_type_attribute: str) → dict[source]

Returns the table rendering options for the given TRLC package, type and attribute.

The returned dictionary may contain the following optional keys:

  • "border" (str): CSS style value applied to the <table> element’s style attribute (e.g. "border: 1px solid black; border-collapse: collapse;").

  • "headingStyle" (str): CSS style value applied to each <th> cell’s style attribute (e.g. "background-color: #c0c0c0;").

Parameters:

  • trlc_package (str) – The TRLC package.

  • trlc_type (str) – The TRLC type.

  • trlc_type_attribute (str) – The TRLC type attribute.

Returns:

Table options dict, or empty dict if no table options are configured.

Return type:

dict
