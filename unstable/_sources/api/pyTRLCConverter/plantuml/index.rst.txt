   .. py:method:: _generate_server(diagram_type: str, diagram_path: str, dst_path: str) -> None

      Generate image using a plantuml server.

      Does not require Java installed and is usually a lot faster
      as no Java startup time is needed when using the plantuml.jar file.

      :param diagram_type: Diagram type, e.g. svg. See PlantUML -t options.
      :type diagram_type: str
      :param diagram_path: Path to the PlantUML diagram.
      :type diagram_path: str
      :param dst_path: Path to the destination of the generated image.
      :type dst_path: str

      :raises FileNotFoundError: PlantUML diagram file not found.
      :raises OSError: Destination path does not exist.
      :raises requests.exceptions.RequestException: Error during GET request to PlantUML server.



   .. py:method:: _generate_to_bytes_local(diagram_type: str, diagram_source: str) -> bytes

      Generate image via local plantuml.jar using -pipe mode and return raw bytes.

      :param diagram_type: Diagram type, e.g. png.
      :type diagram_type: str
      :param diagram_source: PlantUML diagram source text.
      :type diagram_source: str

      :raises FileNotFoundError: Java not installed or plantuml.jar not found.

      :returns: The raw image bytes.
