   .. py:method:: _generate_local(diagram_type, diagram_path, dst_path)

      Generate image local call to plantuml.jar.

      :param diagram_type: Diagram type, e.g. svg. See PlantUML -t options.
      :type diagram_type: str
      :param diagram_path: Path to the PlantUML diagram.
      :type diagram_path: str
      :param dst_path: Path to the destination of the generated image.
      :type dst_path: str

      :raises FileNotFoundError: PlantUML java jar file not found.
      :raises OSError: Destination path does not exist.



   .. py:method:: _generate_server(diagram_type: str, diagram_path: str, dst_path: str) -> None

      Generate image using a plantuml server.

      This is does not require java installed and is usually a lot faster
      as no java startup time needed when using the plantuml.jar file.

      :param diagram_type: Diagram type, e.g. svg. See PlantUML -t options.
      :type diagram_type: str
      :param diagram_path: Path to the PlantUML diagram.
      :type diagram_path: str
      :param dst_path: Path to the destination of the generated image.
      :type dst_path: str

       Raises:
          FileNotFoundError: PlantUML diagram file not found.
          OSError: Destination path does not exist.
          requests.exceptions.RequestException: Error during GET request to PlantUML server.
