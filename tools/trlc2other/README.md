# trlc2other

The scripts ```make_markdown.[bat|sh]``` will generate Markdown files from the requirements and test cases to the ```./out/``` folder.

The scripts ```make_rst.[bat|sh]``` will generate reStructuredText files from the requirements and test cases to the ```./out/``` folder.

The scripts ```make_docx.[bat|sh]``` will generate a docx file from the requirements and test cases to the ```./out/``` folder.

The scripts ```make_reqif.[bat|sh]``` will generate ReqIF files from the requirements and test cases to the ```./out/``` folder.
